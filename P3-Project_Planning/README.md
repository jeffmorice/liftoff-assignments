### Wireframes

* [As a consumer of diverse forms of media, I want to be able to store all of my anticipated titles of different types in one place so I can keep track of them.](https://drive.google.com/open?id=1TyGd96FasgyLl-M-2hrnBOsEIvPCeVg0)
* [As an indecisive user with an overwhelming backlog, I just want random suggestions of media to consume next, so I can reduce my backlog.](https://drive.google.com/open?id=1sQkbWXRmw7Bnrj2DW6IgCmvOViTggne4)
* [As a human user, I want a suggestion app that takes me and my state of mind into account, so I can feel I am receiving non-random suggestions.](https://drive.google.com/open?id=1qhE8gZMeD68UZggFx6YbsSP_LMpuBZxX)
* [As an indecisive user with less free time, I want to recieve smart suggestions of media to consume next, so I can get the most enjoyment out of my leisure time.](https://drive.google.com/open?id=1Kbs-s8INdifWoqU97MfLhvmYkE8Wg-LM)
